Is a WordPress plugin and Elementor page builder extension with Material Design widgets and icons.

== Description ==

= MATERIAL DESIGN =

Material is an adaptable system of guidelines, components, and tools that support the best practices of user interface design. Backed by open-source code, Material streamlines collaboration between designers and developers, and helps teams quickly build beautiful products.

= MATERIAL DESIGN ICONS =

They are simple, modern, friendly, and sometimes quirky icons. Each icon is created using our Material Design guidelines to depict in simple and minimal forms the universal concepts used commonly throughout a UI.
Materializor integrates v7.0.96 (7134 icons) with Elementor icon library.

= WIDGETS AVAILABLE =

- [**Button**](https://runcoders.net/materializor/button/). Buttons allow users to take actions, and make choices, with a single tap.
- [**Card**](https://runcoders.net/materializor/card/). Cards contain content and actions about a single subject.
- [**Chip**](https://runcoders.net/materializor/chip/). Chips are compact elements that represent an input, attribute, or action.
- [**Collapsible**](https://runcoders.net/materializor/collapsible/). Collapsibles are accordion elements that expand when clicked on.
- [**Fixed Action Button**](https://runcoders.net/materializor/fixed-action-button/). A fixed floating action button that allows multiple actions for a screen.
- [**Heading**](https://runcoders.net/materializor/heading/). Headings have predetermined typography and style.
- [**Progress**](https://runcoders.net/materializor/progress/). Progress express an unspecified wait time or display the length of a process.
- [**Spinner**](https://runcoders.net/materializor/spinner/). Spinner express an unspecified wait time of a process.
- [**Tabs**](https://runcoders.net/materializor/tabs/). Tabs organize content across different screens, data sets, and other interactions.

== Installation ==

= Minimum Requirements =

* WordPress 5.0 or greater
* PHP version 7.0 or greater

= Installation =

1. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents in the `wp-content/plugins/` directory of your WordPress installation.
2. Activate the plugin through the 'Plugins' menu in WordPress.
